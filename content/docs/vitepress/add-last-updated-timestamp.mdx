# Add "Last Updated Timestamp"

## Edit `config.mjs`

We will be adding code inside this:

```
import { defineConfig } from "vitepress";

export default defineConfig({
    // ==> Inside Here <==
});
```

Just add a `lastUpdated` prop inside the `defaultConfig`.

```
import { defineConfig } from "vitepress";

export default defineConfig({
    lastUpdated: true // [!code ++]
});
```
